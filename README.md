# Landing Page

A responsive "Coming Soon" landing page built with React.

## Installation

```bash
npm install
```

## Development

Start the development server:

```bash
npm start
```

The page will open at [http://localhost:3000](http://localhost:3000).

## Build

Create a production build:

```bash
npm run build
```

## Features

- Fully responsive design
- Works on desktop, tablet, and mobile
- Built with React
- No external dependencies (except React and React DOM)

## Project Structure

```
src/
├── App.js          # Main component with illustration and text
├── App.css         # Styling with responsive breakpoints
└── index.js        # React entry point

public/
└── index.html      # HTML template

package.json        # Project dependencies and scripts
.gitignore          # Git ignore rules
```
